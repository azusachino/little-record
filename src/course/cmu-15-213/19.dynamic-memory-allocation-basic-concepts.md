# Dynamic Memory Allocation: Basic Concepts

Dynamic memory allocators manage an area of process virtual memory known as the `heap`

## The malloc package

```c
#include <stdlib.h>

void *malloc(size_t size);
void free(void *p);
```

**Example**

```c
#include <stdio.h>
#include <stdlib.h>

void foo(int n) {
    int i, *p;
    p = (int*) malloc(n * sizeof(int));
    if (p == NULL) {
        perror("malloc");
        exit(0);
    }
    for (i = 0; i < n; i++) {
        p[i] = i;
    }
    free(p);
}
```
